clan_composition_age-sex_distribution_Morgane : script to get the average age distribution and 
						sex ratio in the three studied clans over 1989-
						2019.

GLM_functions_calculate_predictors : script with all the functions written to process the raw data into:
                                     - segments with a given number of roadkills each
                                     - covariates (segment road type, percentage woodland cover)

GLM_data_calculation_&_processing : script using the functions for the above script.

monitoring_search_effort_visualization : visualization of the monitoring effort

useful_info_manuscript: script used to compute very diverse information used in the manuscript such as
                        sample size for tests, the shift in km between the median location of roadkills
                        in the wet season and in the dry season, etc.
